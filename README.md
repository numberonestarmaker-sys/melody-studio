# Melody Studio Pro 🎼

آهنگساز هوشمند — ضبط صدا با دهان، تشخیص نت، اصلاح با هوش مصنوعی، و دانلود MIDI/WAV.

## ✅ راه‌اندازی روی GitHub Pages (۲ دقیقه)

### مرحله ۱ — ایجاد ریپازیتوری
1. به [github.com/new](https://github.com/new) برو
2. یک نام بده، مثلاً `melody-studio`
3. **Public** انتخاب کن (برای Pages رایگان لازم است)
4. روی **Create repository** کلیک کن

### مرحله ۲ — آپلود فایل‌ها
1. در صفحه ریپازیتوری، روی **"uploading an existing file"** کلیک کن
2. همه فایل‌های این پوشه را drag & drop کن:
   - `index.html`
   - `app.js`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. پایین صفحه روی **Commit changes** کلیک کن

### مرحله ۳ — فعال‌سازی GitHub Pages
1. در ریپازیتوری برو به **Settings**
2. از منوی سمت چپ **Pages** را انتخاب کن
3. زیر **Source**، شاخه (Branch) را روی **main** بگذار و پوشه را **/ (root)** انتخاب کن
4. **Save** بزن

### مرحله ۴ — صبر کن و باز کن
بعد از ۱-۲ دقیقه، آدرس سایتت این شکلی می‌شود:
```
https://USERNAME.github.io/melody-studio/
```
(جای USERNAME نام کاربری گیت‌هابت را بگذار)

این آدرس **HTTPS** است، یعنی میکروفون بدون مشکل کار می‌کند. ✅

## 📱 نصب روی گوشی (بعد از باز کردن لینک)

**iOS Safari:**
دکمه Share (□↑) → Add to Home Screen

**Android Chrome:**
منوی ⋮ → Install App / Add to Home Screen

بعد از نصب، آیکون روی صفحه اصلی گوشی ظاهر می‌شود و مثل یک اپ واقعی کار می‌کند — بدون نوار آدرس مرورگر.

## 🎯 ویژگی‌ها

- **🎤 ضبط صدا** — زمزمه کن، نت‌ها به‌صورت زنده تشخیص داده می‌شوند
- **🎹 پیانو تعاملی** — نت‌های گام انتخابی هایلایت می‌شوند
- **✦ اصلاح با AI** — نت‌های فالش به نزدیک‌ترین نت گام تبدیل می‌شوند
- **🎼 ساخت آهنگ کامل** — AI از موتیف تو یک قطعه کامل می‌سازد
- **🎲 ۱۰ ملودی مشابه** — تغییرات کوچک روی همان ملودی، نه سبک‌های بیرونی
- **⬇ دانلود MIDI & WAV** — برای استفاده در GarageBand، FL Studio، یا هر DAW

## ⚠️ نکات مهم

- این اپ نیاز به **اتصال اینترنت** دارد (برای بخش‌های AI)
- میکروفون **فقط روی HTTPS** کار می‌کند — این محدودیت امنیتی مرورگرهاست، نه باگ
- اگر فایل را مستقیم (`file://`) باز کنی، میکروفون کار نخواهد کرد — حتماً GitHub Pages یا هاست مشابه استفاده کن
